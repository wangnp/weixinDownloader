# 微信公众号文章批量下载工具
无需安装证书，**监听100%成功**，**链接监听秒级抓取**，支持Win10及以上版本

目前工具已支持：**单个文章下载、首页合集下载、标签合集下载、历史文章批量下载**

支持多格式导出：**word、pdf、html、markdown**

支持：**关键词筛选、下载页数控制、日期范围控制、延时设置**

支持**评论导出**
